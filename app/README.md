# AI Agent 06: Brand Intelligence Dashboard

브랜드의 시장 구조적 흐름과 소비자의 정성적 목소리를 결합하여 데이터 기반 의사결정 체계를 구축하는 AI Agent입니다.

## 주요 기능

### M1: Brand 경쟁 구도 & Market Landscape 분석

#### M1-1: Real Exposure Breadcrumb Mapping (실노출 경로 매핑)
- 플랫폼 내 제품의 모든 계층적 노출 경로 추적
- 명목 카테고리와 실제 노출 카테고리 간 Gap 분석
- 트래픽, 순위, 전환율을 경로별로 시각화

#### M1-2: Volatility Index Tracking (시장 변동성 지수)
- 카테고리별 상위 30위권 브랜드 진입/이탈 빈도 추적
- 실시간 변동성 지수로 시장 재편 시그널 포착
- 주간 변동성 추이 차트로 트렌드 확인

#### M1-3: Reference Brand Sensor (신생 브랜드 벤치마킹)
- 급상승 중인 Emerging Brand 식별
- 성공 브랜드의 카테고리 점유 전략 및 키워드 분석

### M2: Review Intelligence - Consumer Needs Mapping

#### M2-2: Usage Context Analysis (소비 맥락 분석)
- 제품 사용 맥락(시간, 장소, 피부 고민) 데이터 추출
- 구체적 타겟팅 시나리오 도출

#### M2-3: Intelligence Bridge (전략 연계)
- M1(시장 구조)과 M2(소비자 언어) 데이터 결합
- 카테고리 최적화 전략 제안
